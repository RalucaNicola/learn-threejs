Fun project whose only goal was to learn some more 3D. Made with ThreeJS. I used Delaunator for computing the triangles for the irregular triangulated network. Glass reflection photo by Paweł Czerwiński on Unsplash. Pepper the penguin was created by Damon Pidhajecky and she normally lives on Google Poly. The whole project was built with Parcel, a tiny and super easy to use application bundler. The code is available on Github.
